This example solves a 1D problem and compares against existing simulators (E300, AD-GPRS) to verify correctness.

The case is loaded from an input file that can be run in other simulators. For convenience, we provide solutions from the other simulators as a binary file.

julia
using JutulDarcy
using Jutul
using GLMakie
dpth = JutulDarcy.GeoEnergyIO.test_input_file_path("SIMPLE_COMP")
data_path = joinpath(dpth, "SIMPLE_COMP.DATA")
case = setup_case_from_data_file(data_path)
result = simulate_reservoir(case)
ws, states = result;
